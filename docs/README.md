# About tutorial

This tutorial was created with [docsify.js](https://docsify.js.org)

## Initialization
To create documentation with **docsify.js** install `docsify-cli`:
```
npm i docsify-cli -g
```
Initialize documentation in `/docs` directory.
```
docsify init ./docs
```
To preview documentation run:
```
docsify serve docs
```
